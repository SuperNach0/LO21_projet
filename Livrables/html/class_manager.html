Detailed Description

template<class T>
class Manager< T >

Classe mère Template abstraite Managers.

Cette classe regroupe les attributs et les méthodes communes aux différents Managers. Elle est composée d'un vector de template<classe T> qui stockera et manipulera les objets T, et d'un chemin de fichier (par défaut) qui mène au fichier .xml dans lequel les objets seront sauvegardés.
