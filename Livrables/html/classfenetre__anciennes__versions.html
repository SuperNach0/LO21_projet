Detailed Description

Classe Fenetre d'affichage des anciennes versions.

Cette fenêtre sert à afficher, sélectionner et éventuellement restaurer une version d'une note. Les versions sont affichées selon la note depuis laquelle le menu contextuel et l'action "Anciennces versions" a été activée.
